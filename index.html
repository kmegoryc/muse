Muse
Muse is an online music visualizer that uses real-time song data (fft, amplitude), and user motion data (leap motion) as inputs.
How To Set Up

  • Step 1:

    Download the newest version of Processing. Run the muse-leap.pde file uploaded here: https://github.com/kmegoryc/leap

  • Step 2:

    Select a sketch of your choice below.

  • Step 3:

    Select a song of your choice in the upper-left corner of the screen.
